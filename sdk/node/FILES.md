# sdk/node/

The `@evo-hq/evo-agent` Node SDK. Zero dependencies, Node 18+, ES modules.

| File | Purpose |
|------|---------|
| `package.json` | npm package config. Version must match the CLI. |
| `README.md` | User-facing SDK docs (install + usage). |
| `LICENSE` | Apache 2.0 license. |

| Subdir | Purpose |
|--------|---------|
| `src/` | Source: `run.js`, `gate.js`, `backend.js`, public exports in `index.js`. |
| `test/` | Tests using Node's built-in `node:test` runner. Run with `npm test`. |
